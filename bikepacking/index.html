GitHub Verbindung
Repository & Token einrichten (einmalig)
Einmalige Einrichtung — Repo-URL reicht zum Lesen, Token nur zum Bearbeiten:
1. Trage die Repo-URL ein (z.B. max/bikepacking-tour)
2. Für Admin-Zugriff (Schreiben): Token unter github.com → Settings → Developer Settings → Personal Access Tokens erstellen
3. Token mit Scope repo (Classic) oder Contents: Read & Write (Fine-grained)
📸
Foto-Tagebuch einrichten
Cloudinary – kostenlos, 25GB Speicher
Einmalige Einrichtung:
1. cloudinary.com ↗ → kostenlos registrieren
2. Dashboard → Cloud Name notieren
3. Settings → Upload → Add upload preset → Signing Mode: Unsigned → Save

GPX von Komoot importieren
Route analysieren & bei Bedarf splitten
So gehts:
1. Öffne deine Tour auf komoot.de
2. Klicke auf … → Tour exportieren → GPX
3. Lade die Datei hier hoch — die App prüft automatisch ob ein Split nötig ist
📁 GPX-Datei hierher ziehen
oder tippen zum Auswählen
3
POIs via overpass-turbo.eu importieren
Camping · Schutzhütten · E-Bike Ladestationen aus OSM
So gehts:
1. Öffne overpass-turbo.eu ↗
2. Führe die vorbereitete Query für deine Route aus (Camping, Hütten, Ladestationen)
3. Klicke Export → GeoJSON herunterladen
4. Lade die export.geojson hier hoch
✅ Die App filtert automatisch nach Routennähe und erkennt alle POI-Typen
📋 Overpass-Query anzeigen / kopieren
🗺️ export.geojson hierher ziehen
oder tippen zum Auswählen
⏳ Verarbeite GeoJSON…
⚙️ Suchradius (Abstand zur Route)
Meter — Hütten höher weil sie oft hangseitig liegen

POI-GPX importieren
gpx-utils.de · onroutemap.de — Camping, Werkstätten, Tankstellen
Camping · Werkstätten · Tankstellen mit Kontaktdaten
Lade die GPX-Datei von gpx-utils.de ↗ hoch — die App erkennt das Format automatisch und dedupliziert die Doppeleinträge.
Alternativ funktioniert auch eine onroutemap.de-GPX für Restaurants, Hotels etc.
✅ gpx-utils.de liefert 45% Telefonnummern & 60% Websites direkt mit

oder tippen · gpx-utils.de & onroutemap.de werden erkannt
